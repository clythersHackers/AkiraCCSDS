# AkiraOS Assets

This directory contains brand assets for the AkiraOS project.

## Logo Files

### Primary Logo
- **File:** `logo.png` (transparent background)
- **Format:** PNG
- **Recommended Usage:** Documentation, README files, presentations
- **Colors:** Purple to blue gradient (#7B2FF7 → #4A90E2)

### Full Logo with Background
- **File:** `AKIRAOS-LOGO.png` (with background)
- **Format:** PNG
- **Recommended Usage:** Social media, promotional materials

## Usage Guidelines

### Sizing Recommendations

| Context | Width | Example |
|---------|-------|---------|
| README Header | 300px | Main repository README |
| Documentation Pages | 200-250px | Docs index pages |
| Small Headers | 150px | Section headers |
| Icons | 64-128px | Favicons, app icons |

### Markdown Examples

```markdown
<!-- Large header -->
<img src="assets/logo.png" alt="AkiraOS Logo" width="300"/>

<!-- Medium size -->
<img src="assets/logo.png" alt="AkiraOS Logo" width="200"/>

<!-- Small icon -->
<img src="assets/logo.png" alt="AkiraOS Logo" width="100"/>
```

### HTML Examples

```html
<!-- Centered with link -->
<div align="center">
  <a href="https://github.com/ArturR0k3r/AkiraOS">
    <img src="assets/logo.png" alt="AkiraOS Logo" width="300"/>
  </a>
</div>
```

## Brand Colors

The AkiraOS logo uses a distinctive purple-to-blue gradient:

- **Primary Purple:** `#9B59B6` (RGB: 155, 89, 182)
- **Secondary Purple:** `#7B2FF7` (RGB: 123, 47, 247)
- **Primary Blue:** `#4A90E2` (RGB: 74, 144, 226)
- **Light Blue:** `#5DADE2` (RGB: 93, 173, 226)

## Logo Design

The AkiraOS logo features a dynamic starburst pattern symbolizing:
- **Expansion:** Representing growth and scalability
- **Energy:** Dynamic performance and efficiency
- **Modularity:** Multiple points converging to a unified core
- **Innovation:** Forward-thinking embedded systems design
